# Fan Cleaning

Cleaned the internal cooling system to restore proper airflow.

Process:
- Inspected fan and surrounding area for dust buildup
- Removed dust from fan blades and vents
- Cleared debris from airflow pathways

Observations:
- Significant dust accumulation on fan blades
- Dust buildup restricting airflow

Outcome:
- Improved airflow through the cooling system
- Reduced risk of overheating
- Fan able to operate more efficiently

This step addressed thermal performance issues caused by dust buildup.
