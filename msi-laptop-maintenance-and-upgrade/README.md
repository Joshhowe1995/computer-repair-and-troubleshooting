# MSI GF63 Laptop Maintenance, Cleaning and Hardware Upgrade

## Overview

This project documents the disassembly, inspection, internal cleaning, hardware upgrades, and post-repair troubleshooting of an MSI GF63 laptop.

The objective was to diagnose thermal and performance issues, restore proper airflow, and improve system performance through maintenance and component upgrades.

---

## Issue Identified

- Significant dust buildup inside the cooling system  
- Restricted airflow affecting thermal performance  
- Reduced battery reliability requiring replacement  
- General performance limitations due to hardware constraints  
- Boot failure after hardware changes due to missing or corrupted bootloader  

---

## Work Performed

- Laptop disassembly  
- Internal hardware inspection  
- Installed RAM (2 × 4GB Samsung modules in dual-channel configuration)  
- Identified and removed M.2 SSD  
- Replaced internal battery  
- Analysed cooling system  
- Removed dust and restored airflow  
- Reassembled device and performed testing  
- Diagnosed boot failure and restored system using bootable USB installation  

---

## Key Findings

- Dual RAM slots available and successfully upgraded  
- M.2 SSD installed and removable  
- Battery was replaceable and successfully upgraded  
- Heavy dust accumulation on cooling fans  
- Airflow significantly restricted prior to cleaning  
- Bootloader was missing or corrupted after hardware changes  

---

## Outcome

- Increased system memory (8GB total RAM)  
- Improved multitasking and responsiveness  
- New battery installed improving reliability  
- Internal components cleaned and restored  
- Improved cooling efficiency  
- Reduced risk of overheating  
- System boot functionality restored via OS reinstallation  
- Device powered on and functioned correctly after full repair
